# Contributing

Thanks for the help! If you are interested to write some code for this repo:
1. Fork this repo and switch on the branch that will be auto generated from Project
2. Test locally before open the PR
3. Open a PR to main branch 
4. Pr must:
   - success CI
   - have the review of CODEOWNERS
   - resolve all conversation
5. Be sure to not push any secrets or env variables
